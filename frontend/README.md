# AI Avatar Stream - Frontend

React dashboard for controlling the AI discussion stream.

## Development

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Environment Variables

Create `.env`:

```
VITE_API_URL=http://localhost:8000
```
